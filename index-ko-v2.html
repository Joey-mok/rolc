현직 DIT·편집자가 실무를 위해 개발한 초경량 뷰어

편집 들어가기 전에,
소스부터 빠르게 정리하세요.

ROLC(Roll Change)는 촬영 직후 대량의 ProRes / MP4 클립을 순식간에 로드해 레이팅하고, 단 몇 번의 클릭만으로 편집팀에 인계할 완벽한 PDF 샷 리포트를 뽑아내는 독보적인 실무 툴입니다.

↓ 무료 체험판 다운로드 (Universal macOS) 라이선스 평생 소장권 구매

오프라인 RSA 인증 방식 · 1회 결제 평생 소장 · 기기 귀속 지원

Features

필요한 기능만.
군더더기 없이.

현장에서도, 편집실에서도 밤샘 작업을 줄이고 퇴근 시간을 앞당기기 위해 설계됐습니다.

🎞

썸네일 그리드 — 한눈에 전체 소스

NAS나 외장 SSD 폴더를 여는 즉시 수백 개의 클립이 썸네일 그리드로 펼쳐집니다. 고속 썸네일 캐싱 탑재로 무거운 프로그램 없이 마우스 스크러빙으로 내용물을 즉시 확인합니다.

⌨️

키보드 단축키 등급 분류 (O/K/X)

클립을 재생하며 키보드 'O'(OK), 'K'(Keep), 'X'(Reject), 'U'(Unrated) 단 한 글자 입력으로 등급을 매깁니다. 마우스에 손을 대지 않는 초인적 속도를 경험하세요.

🔍

하위 폴더 Deep Scan & 정렬

날짜/장소/카메라/XDROOT/Clip 같이 깊게 숨겨진 폴더도 'Deep Scan' 한 번에 3단계 깊이 제한 없이 완전 재귀 탐색하여 영상 클립을 빠짐없이 가져옵니다. 특정 카메라만 골라볼 수도 있습니다.

📄

PDF 샷 리스트 리포트 출력

레이팅 완료 즉시 깔끔한 그리드형 또는 상세 리스트형 PDF 리포트를 내보냅니다. 사이드바 폴더 이름변경 기능과 연계되어, PDF 헤더에 변경된 폴더명이 그대로 박힙니다.

📦

프로젝트 저장 및 불러오기 (.cvp)

레이팅 결과를 가벼운 .cvp 프로젝트 파일로 저장해 동료나 타 편집실 Mac에 전달하세요. 클라우드나 DB 서버가 없어도 완벽한 협업 연동을 지원합니다.

🔒

강력한 RSA 오프라인 라이선스

인터넷이나 휴대전화 수신이 차단되는 지하 스튜디오, 숲속 야외 로케이션, 해외 오지 현장에서도 완전 오프라인으로 라이선스 키 등록과 프로그램 실행이 완벽 작동합니다.

목조이 (Joey Mok)

ROLC 개발자 / 현직 DIT & 편집자

Developer's Story

"현장의 비효율에 답답해 직접 코딩했습니다"

"숲속 촬영지나 전파가 끊기는 터널 안, 하루에만 수백 기가의 ProRes422 대용량 클립이 쏟아집니다. 메인 감독에게 보고할 OK컷 분류를 위해 연 120만 원이 넘는 Silverstack을 구독하기는 너무 버겁고, 단순 뷰어인 Kyno는 업데이트가 끊겼습니다. 보조 편집자들과 조연출은 매번 무거운 NLE 프로그램을 켜고 밤새 일일이 더블클릭하며 지쳐 가고 있었습니다."

그 뼈아픈 현장 경험을 바탕으로, 1인 개발자로 복귀하여 오직 실무자를 위한 초경량 뷰어를 개발했습니다. 현장의 은어이자 핵심 소통 절차인 '롤체(Roll Change)'에서 이름을 딴 ROLC는, 단축키 3개와 한글 PDF 리포트 생성이라는 가장 단순하고 날카로운 무기만을 담고 있습니다.

과도한 기능으로 무거워지거나 고가의 정기 구독료를 요구하지 않습니다. 오직 편집팀의 빠른 퇴근과 현장의 가벼운 워크플로우를 보장하는 단 하나의 파트너가 되겠습니다.

Rating System

세 가지 등급.
명확한 선택본.

실제 영화/드라마 컷편집 워크플로우에서 가장 본질적으로 쓰이는 단순하면서 강력한 분류 체계입니다.

O

OK — 우선 사용컷 (A-Roll)

본 촬영의 최고의 테이크. 메인 편집 감독과 연출자에게 최우선으로 보여줄 확실한 테이크입니다.

K

Keep — 대안컷 (B-Roll)

구도나 연기가 괜찮아 대안으로 쓸 수 있는 백업 테이크입니다. 오케이컷은 아니지만 버리기 아까운 소스입니다.

X

Reject — 불용컷 (Trash)

포커스가 나갔거나, 불필요한 테이크, 슬레이트 오작동 등 편집에 제외할 소스입니다. 눈앞에서 숨겨 작업의 집중도를 높입니다.

macOS Installation Tip

처음 실행 시 "확인되지 않은 개발자" 경고가 뜨나요?

ROLC는 보안 인증 비용을 배제하여 고객 단가를 획득하기 위해, 초기 설치 시 애플 공증 서명을 생략했습니다. 해킹 프로그램이 아니니 안심하고 아래 방법 중 하나로 실행하세요.

A 간편 우클릭 열기 (권장)

가장 단순하고 안전하게 게이트키퍼를 우회하는 방법입니다. 최초 1회만 설정하면 다음부터는 더블클릭으로 바로 켜집니다.

  1. 응용 프로그램(Applications) 폴더 안의 ROLC 아이콘을 우클릭(또는 Control+클릭)합니다.
  2. 메뉴에서 "열기"를 선택합니다.
  3. "Apple이 악성 소프트웨어 여부를..." 팝업이 뜨면 "열기" 버튼을 다시 한 번 클릭합니다.

B 터미널 명령어로 공증 보안 해제

보다 완벽한 앱 실행 차단을 풀고 싶은 사용자, 또는 게이트키퍼 경고가 계속해서 뜨는 경우 터미널을 이용해 명시적으로 해제할 수 있습니다.

  1. 맥의 '터미널(Terminal.app)' 프로그램을 켭니다.
  2. 아래 명령어를 통째로 복사한 뒤 터미널 창에 붙여넣고 Enter를 탭합니다.
    3. xattr -cr /Applications/ROLC.app
  3. 이제 ROLC 앱을 응용 프로그램 폴더에서 손쉽게 더블클릭하여 바로 가동합니다.

영상 편집자·DIT를 위한 macOS 앱

편집 들어가기 전에,
소스부터 정리하세요.

ROLC는 촬영 완료된 소스를 빠르게 훑고, OK·Keep·Reject로 분류해 선택본을 즉시 뽑아내는 클립 리뷰 도구입니다. 편집 전 사전 작업 시간을 절반으로 줄여드립니다.

macOS · 완전 오프라인 · 유니버설 바이너리

Features

필요한 기능만.
군더더기 없이.

현장에서도, 편집실에서도 빠르게 움직여야 하는 영상 전문가를 위해 설계됐습니다.

🎞

썸네일 그리드 — 한눈에 전체 소스

촬영 폴더를 열면 모든 클립이 썸네일로 즉시 표시됩니다. 플레이어를 열지 않고도 수백 개의 클립을 훑어볼 수 있습니다.

⌨️

키보드 한 번으로 등급 분류

O·K·X 키 하나로 클립을 즉시 등급 분류합니다. 리듬을 끊지 않고 빠르게 전체 소스를 검토할 수 있습니다.

🔍

등급·카메라별 필터

OK 클립만 즉시 골라내거나 특정 카메라 소스만 집중해서 볼 수 있습니다. 다중 폴더 사이드바로 멀티캠 현장도 문제없습니다.

📄

PDF 샷 리스트 내보내기

선택한 클립 목록을 PDF 샷 리스트로 즉시 출력합니다. 편집감독·감독·클라이언트에게 바로 공유 가능합니다.

📦

프로젝트 저장 및 불러오기

분류 작업을 파일로 저장하고 언제든 다시 불러옵니다. 데이터베이스도, 클라우드도 필요 없습니다. 파일 하나로 관리합니다.

🔒

완전 오프라인 작동

인터넷 없는 현장에서도 완벽하게 작동합니다. 계정 생성, 클라우드 연결, 인터넷 연결이 일절 필요 없습니다.

Rating System

세 가지 등급.
명확한 선택본.

실제 편집 워크플로우에서 자연스럽게 쓰이는 분류 체계입니다.

O

OK — 우선 사용컷

반드시 쓸 베스트 테이크. 감독·편집감독에게 먼저 보여줄 클립입니다.

K

Keep — 대안컷

OK가 안 될 경우 대안으로 쓸 수 있는 컷. 충분히 쓸 만하지만 최선은 아닌 테이크.

X

Reject — 불용컷

핀트 아웃, 노출 실패, 음향 불량 등 쓸 수 없는 컷. 표시 후 시야에서 제거합니다.

How It Works

메모리 카드 옮긴 후
선택본 완성까지 단 3단계.

설치하고 바로 씁니다. 계정 생성도, 설정도 필요 없습니다.

1

폴더 열기

촬영 소스 폴더를 ROLC에 지정합니다. 캐시된 미리보기를 활용해 대용량 소스도 썸네일을 즉시 불러옵니다.

→
2

키보드로 등급 분류

O·K·X 키로 등급을 매기고, 방향키로 클립을 이동합니다. 분류 결과는 자동으로 저장됩니다.

→
3

필터·내보내기·공유

OK컷만 골라내 PDF 샷 리스트로 내보내 편집감독이나 감독에게 바로 전달합니다.

System Requirements

최신 macOS 환경에
최적화됐습니다.

유니버설 바이너리로 Apple Silicon과 Intel Mac 모두에서 네이티브 성능으로 실행됩니다. Rosetta 불필요.

운영체제 macOS 12 Monterey 이상
프로세서 Apple Silicon 네이티브 Intel 네이티브
메모리 8GB RAM 권장
저장 공간 300MB 이상
인터넷 연결 불필요 완전 오프라인
지원 포맷 .mov · .mp4 · .m4v · .mxf

DEVELOPER NOTES

"Built out of sheer frustration on set"

"While configuring workflows and offering technical assistance to drama editing teams, I watched assistants struggle daily, verifying and marking over a hundred raw clips inside Premiere Pro. I realized there was a critical need for a lightweight tool where I could instantly playback clips using just a spacebar, visually scrub thumbnails, rate/filter clips quickly, and import cleanly into Premiere."

That pain drove me to step back and craft a targeted desktop utility. Named after 'Roll Change' — the actual camera card-swapping process on set — ROLC packages exactly what you need to cut out the logging delay: fast ratings, elegant PDF sheets, and simple .cvp data sharing.

We do not weigh down your Mac with unnecessary bloat or trap you in expensive, endless subscription fees. ROLC's sole purpose is saving you logging hours and letting you head home early.

macOS Installation Tip

Does macOS block execution with an "Unverified Developer" warning?

To keep prices extremely low for freelancers, ROLC doesn't carry expensive Apple certification signatures in early builds. We are NOT malicious malware. Please bypass Apple Gatekeeper using one of these easy steps:

A Right-Click Open Bypass (Recommended)

The simplest and standard way to bypass Gatekeeper. This needs to be done only once, and after that, the app launches with normal double-clicks.

  1. Open your Applications folder and find ROLC.app.
  2. Right-click (or Control+Click) on the icon and select "Open".
  3. When the unverified warning pop-up displays, click "Open" again to authorize execution.

B One-Line Terminal Command

If you prefer using developer shell or Gatekeeper warnings keep persisting, run this single clean xattr line in Terminal to release security attributes.

  1. Open your Mac's default Terminal (Terminal.app).
  2. Copy and paste the exact command line below, and tap Enter:
    3. xattr -cr /Applications/ROLC.app
  3. ROLC is now fully authorized to launch on your machine via standard double-clicks.
